# 独角数卡西瓜版 v3.0.0-alpha.1

发布日期：2026-04-12

版本定位：

- 首个现代化重构预发布版本
- 面向内部验证、开发环境试用和迁移路线确认
- 不是稳定正式版，不建议直接作为长期生产基线

本版重点：

- 恢复并固定遗留运行时基线，建立可重复的本地开发和 CI 验证路径
- 建立核心业务 PHPUnit 护栏，当前主线测试结果为 `OK (211 tests, 812 assertions)`
- 去掉对常驻队列守护进程的硬依赖，订单过期改为调度扫描
- 让安装主路径切换到 `migrate + bootstrap seed + 显式创建管理员`
- 清理多条升级阻塞依赖链，包括 `prophecy`、`geetest`、旧二维码依赖
- 支付层完成第一轮收口，退役 `Paysapi`、`Vpay`、`PayJS`
- `Stripe` SDK 已升级到较新的主版本基线，并完成服务化隔离
- 新默认前台主题切换到 `avatar`
- 新后台壳已经承接多张真实页面，并把 `/admin` 默认入口切到后台壳首页

当前后台壳覆盖：

- Dashboard
- 系统设置概览与多组配置动作页
- 邮件测试动作页
- 商品分类
- 优惠码
- 支付通道
- 商品
- 订单
- 卡密与卡密导入
- 邮件模板

当前已知仍在进行中的主线：

- 旧 Dcat 后台仍未完全退场
- 后台壳还在持续扩容中
- 支付层保留通道仍需继续收口
- 安全治理和 PHP / Laravel 升级前清障尚未进入最终阶段

建议用途：

- 内部验收当前现代化进度
- 作为后续 `beta` 阶段的基准点
- 用于新后台壳、安装链、测试基线和支付层现状的集中评审
